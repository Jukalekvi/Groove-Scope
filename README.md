# Groove Scope
Groove Scope on musiikin analysointityökalu, joka näyttää Spotifyssa kuunnellusta musiikista saatua käyttäjädataa visuaalisessa muodossa.
# Sovellusarkkitehtuuri
## Frontend
Ohjelmisto hyödyntää Next.js-kirjastoa käyttöliittymän rakentamiseen ja datan visualisointiin.
## Backend
Backend on rakettenu Go-ohjelmointikielellä, joka käsittelee Spotify API:sta haettua dataa ja välittää sen frontendille omalla palvelimella.
## Spotify Developer
Sovellus käyttää Spotify Developer -alustaa käyttäjädatan hakemiseen ja analysointiin sinne luodusta sovelluksesta.
